Microarray Analysis Tool

MiCA is a fully integrated, interactive and streamlined Windows desktop-based environment for microarray gene expression analysis. Typical tasks of gene expression analysis such as data fetching, editing/grouping, data pre-processing and differential gene expression are all implemented in an interactive and easy to use interface. A complete list of features currently available with the latest version of MiCA can be viewed at the features tab.

How to install MiCA?

  • Download MiCA_1_1_2_setup.exe and install
  • Launch MiCA from desktop shortcut
  • NOTE: Always run installer and application as administrator
Features

The set of features offered by MiCA eliminates the need to use any external input or application as all major tasks of microarray gene expression analysis can be accomplished from within the application.

  • Project Management with the separate directories
  • Browse GEO datasets by searching for appropriate terms
  • View characteristics for each GEO browser search result
  • Fetch phenotype data through GEO browser or directly by specifying a GEO series accession identifier
  • Fetch raw gene expression data
  • Edit phenotype data using advanced editor that allows grouping, comparisons, character removal and creation of new attributes
  • Change attribute datatypes
  • Support for BrainArray custom CDF
  • Support for rma, gcrma, mas5, dchip & plier normalization methods with parameter customization
  • Support for batch-effect removal using combat, mean-centring, ratio-a and ratio-g
  • Store normalized expression sets to local storage for faster access during multiple startups
  • Widget to select active normalized expression set
  • Visualize data using interactive frequency histograms, intensity histograms, boxplots, PCA, ROBPCA, ma-plot, microarray pictures RLE and NUSE plots
  • Identify and remove outlier with a single click through interactive visualizations
  • Differential gene expression using multiple processing models
  • Highly customizable linear models with support for confounding factors, interaction terms and surrogate variables
  • Generalized linear models with support for confounding factors and surrogate variables
  • Implementation of LIMMA to provide quick differential expression results
  • Use surrogate variable analysis to capture un-indentified variation in the dataset (using manually specified number of surrogate variables or automatically selecting the significant variables)
  • Support for Significance Analysis of Microarrays (SAM) for differential expression
  • Support for gene set enrichment analysis using GSVA (visualize gene sets using heatmaps)
  • Gene set manager to fetch KEGG gene sets or create/import custom gene sets
  • Visualize results using differential gene lists, interactive heatmaps and class-based boxplots
  • Compare gene lists from different studies
  • Generate comprehensive PDF report
  • Widget to view current CPU and memory usage

System Requirements

MiCA uses a large number of published R libraries to support its backend processing & calculations for analysis. Therefore, please make sure that R version 3.4.4 is installed before running the application.

  • Microsoft Windows Operating System 7 or above
  • Dual core processor 1.6GHz+ (capable of running modern x64 applications)
  • 1024 x 768 or above resolution
  • Minimum 4GB of memory (8GB recommended)
  • Minimum 100MB of disk space (1-2GB recommended for storing raw expression files)
  • Active internet connection
  • R version 3.4.4
  • RTools version 35
